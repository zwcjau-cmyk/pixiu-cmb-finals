# 貔貅学长理财知识库

## 一、新手理财三大铁律

### 铁律一：实盘验证
推荐的基金博主必须满足：
- 实盘操作资金 > 100万
- 年风险回撤 < 12%
- 稳定运作 > 3年
不满足以上任何一条，一律劝退。

### 铁律二：本金至上
- 任何情况下，本金安全第一
- 高收益必然伴随高风险，新手应从低风险产品起步
- 建议初始月利润率目标：3%

### 铁律三：延迟满足
- 冲动消费前，等待24小时冷静期
- 大额支出需通过"感谢信结界"确认
- 目标可视化：将梦想清单进度时刻展示

## 二、资金结构建议（大学生版）

### 基础比例（生活费 2500 元/月为例）
- 固定开支（40%）：1000元 - 房租/饭卡/交通
- 灵活开支（20%）：500元 - 社交/娱乐/零食
- 攒钱池（40%）：1000元 - 目标储蓄 + 理财学习

### 预算管理原则
- 大类不超 5 个，追求简洁
- 允许±10%浮动，不必精确到分
- 每月底复盘，下月微调

## 三、定投策略入门

### 什么是定投
定期定额投资，无需择时，分散风险。

### 适合新手的定投方式
1. 选择宽基指数基金（如沪深300、中证500）
2. 设定固定金额（建议不超过月结余的50%）
3. 坚持至少6个月不动

### PE/PB 估值法则
- PE < 历史百分位 30%：低估，可加仓
- PE > 历史百分位 70%：高估，减少定投
- 在中间区域：正常定投

## 四、防韭菜指南

### 常见割韭菜手法
1. "稳赚不赔"的承诺 → 100%骗局
2. 高息揽储（年化>15%）→ 极高风险
3. 朋友圈晒单 → 幸存者偏差
4. "内部消息" → 违法且不可信

### 正确的学习路径
1. 先看经典书籍（《小狗钱钱》《穷爸爸富爸爸》）
2. 再了解指数基金
3. 小额实盘练手（100-500元）
4. 记录投资日记，复盘学习
